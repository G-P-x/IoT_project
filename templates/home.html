API Endpoints

GET /operator/history/<parameter>

Query Parameters:

  • twin_id - (optional) ID of the digital twin
  • limit - (optional) Maximum records to return (default: 200)

POST /commands

Query Parameters:

  • twin_id - (optional) ID of the digital twin

Request Body:

{
  "target": {"sensor_id": "t1", "parameter": "temperature"},
  "issued_by": "operator_id",
  "timeout_ms": 2000
}
